本教程将指导你添加一个新的 [网关组](../key-concepts/gateway-groups.md)。

## 前提条件

1. 安装 [API7 Enterprise](./install-api7-ee.md)。

## 步骤

1. 从侧边导航栏中选择 **网关组**，然后点击 **添加网关组**。
2. 在表单中执行以下操作：
   * **类型** 选择 `API7 网关`。
   * **名称** 填写 `生产网关组`。
   * 点击 **添加**。

继续去 [新增网关实例](./add-gateway-instance.md) 保证你新添加的网关组可以正常使用。
